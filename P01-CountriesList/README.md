# Countries List

Simple app with a list of countries.

## What I learned:

This project helped me learn about the basic use and implementation of:

- Navigation controller;
- View controller;
- Scroll view;
- Table view;
- Storyboard;
- Parsing JSON from a local file;
- How to pass data between screens.
